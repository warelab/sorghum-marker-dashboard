# Sorghum Marker Review Instructions for Curators

Use the marker review dashboard to validate sorghum marker entries and submit feedback:

```text
https://warelab.github.io/sorghum-marker-dashboard/
```

## Review Goal

Please validate whether each marker assigned to your group is correct for inclusion in the sorghum marker database or marker panel.

For each marker, check:

- Standard marker ID
- Trait
- Chromosome
- Coordinate / position
- Original marker name
- Evidence
- Priority

## How to Review Markers

1. Open the dashboard link.

2. In the left `Filters` panel, choose your group from the `Group` dropdown.

3. The review queue will load markers assigned to your group.

4. Review each marker in the queue. The table shows the marker ID, trait, chromosome, coordinate, priority, and review status.

5. Use the `Marker dropdown` or click marker IDs in the queue to inspect a marker in the `Current marker` panel.

6. In the `Current marker` panel, check the marker details, including original name, chromosome, position, priority, and evidence.

7. Select markers for feedback:

- Check individual markers if reviewing one or a few markers.
- Use `Select all visible` if all currently visible markers should receive the same decision.
- Use search and filters first if you only want to select a subset.

8. Enter your name in `Reviewer name`.

9. Choose one decision:

- `Confirm marker`: marker looks correct.
- `Revise details`: marker is useful, but some information needs correction.
- `Remove from panel`: marker should not be included.
- `Question / needs discussion`: marker is uncertain or needs follow-up.

10. Add a comment when needed.

11. Click `Submit feedback`.

## Important Notes

- One submitted response is recorded for each selected marker.
- If you select multiple markers, the same decision and comment will be applied to all selected markers.
- For specific corrections, select only the affected marker or markers and provide a specific comment.
- If all visible markers look correct, use `Select all visible`, choose `Confirm marker`, and submit once.
- The dashboard sends responses to a central Google Sheet through Google Forms.
- You do not need to email a CSV unless requested.

## What to Put in Comments

Please add comments for:

- Corrected trait name
- Corrected chromosome or coordinate
- Corrected original marker name
- Evidence or publication reference
- Reason for removing a marker
- Any uncertainty or question

## Example Comments

Confirmed marker:

```text
Marker information looks correct.
```

Coordinate correction:

```text
Coordinate should be Chr 2: 59,718,620 based on source file.
```

Trait correction:

```text
Trait should be plant height, not stem height.
```

Remove marker:

```text
Remove because marker is duplicated by S100K-SNP-SB02-059718618-G-A.
```

Question:

```text
Need to confirm whether this marker is associated with flowering time or plant height.
```

## Recommended Review Pattern

For careful review:

1. Choose your group.
2. Review one marker at a time.
3. Select the marker.
4. Choose the decision.
5. Add a comment if needed.
6. Submit feedback.

For batch confirmation:

1. Choose your group.
2. Use filters if needed.
3. Use `Select all visible`.
4. Choose `Confirm marker`.
5. Add a short comment such as `Markers reviewed and confirmed.`
6. Submit feedback.
